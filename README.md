# Aplicativos de Probabilidad Básica

**Descripción**: Este proyecto desarrolla Aplicativos de entrenamiento con Cuestionarios de Probabilidad Básica, y un Simulador de proceso estocástico. Estos productos académicos son resultados del espacio Tutorías de Probabilidad ordenados por la Facultad de Ingeniería de la Universidad Distrital Francisco José de Caldas.

- Proyecto Curricular Ingeniería de Sistemas
- Facultad de Ingeniería
- Universidad Distrital Francisco José de Caldas
- Periodos académicos 2025-3 y 2026-1
